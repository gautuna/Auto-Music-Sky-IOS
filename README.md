## Installation

This tool supports both Jailbreak and Non-Jailbreak devices

> **Note:** We strongly advise against using TrollStore or traditional Jailbreak methods, as these can trigger security detection in banking apps and other games. We highly recommend using **KSign** (or similar signing tools) to install the `.deb` file

### How to Install
1. **Download the binary:** Get the latest `.deb` file from our [**Releases**](https://github.com/gautuna/Auto-Music-Sky-IOS/releases) section
2. **Prepare tools:** Purchase a developer certificate and install **KSign** (you can find services at [**muacert.com**](https://muacert.com) or other providers in your region)
3. **Get the app:** Download the clean `Sky.ipa` file from [**decrypt.day**](https://decrypt.day/app/id1462117269)
4. **Sign the App:** 
   - Open KSign, import the `.deb` file and the `Sky.ipa`
   - In the "App" tab, select Sky and choose **Sign**
   - In the "Tweaks" section, tap **Import**, select your `.deb` file, and enable it
5. **Finalize:** Return to the sign menu, tap **Start Signing**, wait for the process to complete, and install the resulting app
